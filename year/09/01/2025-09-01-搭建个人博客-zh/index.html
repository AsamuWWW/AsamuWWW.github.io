从0开始搭建个人独立博客：工具安装篇

这篇文章将详细分享个人独立博客搭建的第一步——工具准备与安装，全程无废话、不跳步，适配 Windows 11（64位）系统，跟着操作即可完成基础环境搭建。

一、准备工具清单

搭建博客需先安装两款核心工具：Node.js（博客框架运行环境）和 Git（代码版本控制与部署工具），具体版本及作用如下：

工具名称 作用说明 适配版本（Windows 11 64位） 下载地址
Node.js 基于 Chrome V8 引擎的 JavaScript 运行时，为 Hexo 博客框架提供运行环境 node-v24.6.0-x64 https://nodejs.org/en/
Git 分布式版本控制系统，用于后续博客代码的管理与部署（如部署到 GitHub Pages） 2.51.0（x64 version） https://git-scm.com/downloads

二、Node.js 安装步骤（Windows 11 64位）

Node.js 是 Hexo 框架的核心依赖，必须优先安装，以下是完整安装流程：

1. 下载 Node.js

  1. 打开 Node.js 官网：https://nodejs.org/en/；

  2. 点击首页显著位置的 “Get Node.js” 按钮（默认下载最新稳定版，本文以 node-v24.6.0-x64 为例）。

2. 开始安装 Node.js

  1. 找到下载完成的安装包 node-v24.6.0-x64.msi，双击运行；

  2. 弹出安装向导后，点击 【Next】 按钮；

  3. 阅读许可协议，勾选 “I accept the terms in the License Agreement”（同意协议），点击 【Next】；

  4. 选择安装路径：默认路径为 C:\Program Files\nodejs\，若需修改（如避免C盘占用过多），点击 【Browse】 选择自定义文件夹，确认后点击 【Next】；

  5. 功能配置页：勾选 “Automatically install the necessary tools…”（自动安装必要工具，避免后续依赖缺失），点击 【Next】；

  6. 等待安装进度完成，最后点击 【Finish】 按钮；

  7. 安装完成后会自动弹出命令行窗口，按两次任意键，随后再 键入 ENTER，至此 Node.js 安装全部完成。

三、Git 安装步骤（Windows 11 64位）

Git 用于后续博客代码的提交与部署（如推送到 GitHub），以下是详细安装与配置流程：

1. 下载 Git

  1. 打开 Git 官网下载页：https://git-scm.com/downloads；

  2. 官网会自动识别系统，点击 “Windows” 对应的下载链接（本文以 2.51.0 x64 version 为例）。

2. 开始安装 Git

  1. 找到下载完成的 Git 安装包（如 Git-2.51.0-64-bit.exe），双击运行；

  2. 弹出安装向导，点击 【Next】 进入路径选择页：可自定义安装路径（如 D:\Program Files\Git），或保持默认，选择后点击 【Next】；

  3. 组件选择页（建议默认配置，无需修改）：绿色框是是否在桌面创建 Git 快捷方式（按需勾选），红框是默认勾选的核心组件（如“Git Bash”“Git GUI”，不建议修改），蓝色框是是否检查 Git 更新、向 Windows 终端添加 Git Bash 配置（按需勾选），确认后点击 【Next】；

  4. 开始菜单文件夹命名页：默认“Git”即可，无需修改，点击 【Next】；

  5. 编辑器选择页：默认“Vim”（Git 自带编辑器，极少用到），直接点击 【Next】；

  6. 分支名配置页：选择 “Let Git decide”（让 Git 使用默认分支名，新手友好），点击 【Next】；

  7. PATH 环境配置页（关键步骤）：推荐选择 “Git from the command line and also from 3rd-party software”（从命令行及第三方软件调用 Git，兼容性更强），点击 【Next】；

  8. SSH 可执行文件选择页：选择默认选项 “Use bundled OpenSSH”（Git 自带的 OpenSSH 更稳定），点击 【Next】；

  9. HTTPS 后端传输选择页：选择 “Use the native Windows Secure Channel library”（使用原生 Windows 安全通道库，适配 Windows 系统），点击 【Next】；

  10. 行尾符号转换配置页：选择 “Checkout Windows-style, commit Unix-style line endings”（Windows 系统推荐设置，避免换行符兼容问题），点击 【Next】；

  11. 终端模拟器配置页：选择默认选项 “Use MinTTY (the default terminal of MSYS2)”（Git Bash 默认终端，体验更优），点击 【Next】；

  12. 后续页面无需额外配置，均点击 【Next】 或 【Install】，等待安装完成后点击 【Finish】 即可。

四、安装验证（可选，确保工具可用）

安装完成后，可通过以下步骤验证工具是否正常运行：

  1. 按下 Win + R，输入 cmd 打开命令提示符；

  2. 输入 node -v，若显示 Node.js 版本号（如 v24.6.0），说明 Node.js 安装成功；

  3. 输入 git --version，若显示 Git 版本号（如 git version 2.51.0.windows.1），说明 Git 安装成功。

至此，博客搭建的基础工具已全部准备完毕，下一步可开始安装 Hexo 框架并初始化博客。
